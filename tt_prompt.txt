Bạn là một chuyên gia Khoa học Thể thao chuyên về môn Bóng Bàn. Hãy đọc các tài liệu đã nạp và trích xuất dữ liệu sau dưới định dạng JSON duy nhất. KHÔNG trả lời thêm bất kỳ text nào ngoài JSON.

Cấu trúc JSON mong muốn:
{
  "table_tennis_base_muscles": ["các nhóm cơ chính chịu tải lớn nhất"],
  "table_tennis_multipliers": {
    "offensive": { "tên cơ": hệ số nhân (vd 1.2) },
    "defensive": { "tên cơ": hệ số nhân (vd 1.15) },
    "all_round": { "tên cơ": 1.0 },
    "format_doubles_intensity": hệ số cường độ khi đánh đôi (vd 0.85)
  },
  "cns_fatigue_multiplier": hệ số mỏi thần kinh trung ương (vd 1.3),
  "muscle_damage_multiplier": hệ số vỡ cơ (vd 0.8 do bóng bàn ít vỡ cơ hơn),
  "injury_half_life_multipliers": {
    "tên khớp/cơ dễ chấn thương (vd elbows, knees, ankles, wrists)": hệ số làm chậm phục hồi (vd 1.25)
  }
}

Lưu ý: Chỉ sử dụng các keys chuẩn này cho tên cơ: quadriceps, hamstrings, glutes, calves, ankles, knees, obliques, lower_back, lats, front_shoulders, rear_shoulders, biceps, triceps, forearms, wrists, elbows.
